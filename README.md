# Simple_Calculator
A lightweight calculator.  
How to set up:  
This project is built with JDK 17.  
Go to https://www.oracle.com/java/technologies/javase/jdk17-0-13-later-archive-downloads.html to download JDK 17.  
If JDK 17 has already existed on your computer,please ignore this step.  
Download Calculator.class and just run it.
Calculator.java is the source code.
